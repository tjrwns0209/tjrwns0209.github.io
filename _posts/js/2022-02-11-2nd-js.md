# 자바스크립트 변수

+ 변수들

```javascript
var
let
const
```

var와 let은 가변형 변수입니다. 나중에 필요할 때 지정했던 값을 다시 바꿔줄 수 있는 변수입니다. 이와 반대로 const는 불변형 변수로 한 번 지정한 값을 절대로 바꾸어줄 수 없습니다.


+ var를 사용하기 보다는 상황에 따라서 let과 const를 쓰는 것을 추천합니다.
